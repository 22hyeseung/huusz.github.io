Middleware

1. Express

  • Node.js 생태계에서 가장 널리 쓰이는 웹 프레임워크
  • 내장하고 있는 기능은 매우 적으나, 미들웨어를 주입하는 방식으로 기능을 확장하는 생태계를 가지고 있음

2. Express 앱의 기본 구조

1) Routing

2) Request 객체

  • req.body
    요청 바디를 적절한 형태의 자바스크립트 객체로 변환하여 이곳에 저장 (body-parser 미들웨어에 의해 처리됨)
  • req.ip
    요청한 쪽의 IP
  • req.params
    route parameter
  • req.query
    query string이 객체로 저장됨
